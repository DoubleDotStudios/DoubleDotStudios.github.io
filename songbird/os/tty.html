TTY Install

A TTY install is not recommended for most users.


A TTY install is great for minimal systems that need a bit of headway. You can still get all the same work done just without the distraction of a GUI. You can choose between Neovim, Micro, Nano, Vim and Emacs as your editor and Yazi or Ranger as your file manager.


After a TTY install you can always install a GUI environment. This can be useful for installing a Desktop Environment not included in the installer. For example, installing MATE with ly.
