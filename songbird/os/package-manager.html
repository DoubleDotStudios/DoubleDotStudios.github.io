Package Manager

By default, SongbirdOS comes with Pacman and SPM. Pacman is the Arch Linux package manager that's used for official repository packages. SPM is the Songbird Package Manager. It's used instead of Yay or Paru as an AUR helper. This allows you to install packages from the main repository or the Arch User Repository.


If you would rather use Yay or Paru over SPM then this can easily be done in the installer. Simply change the option under Package Manager to your desired choice.
