SongbirdOS

SongbirdOS is an Arch Linux based distribution that aims to be user friendly and customaisable. With the option to use a GUI installer or the TTY, the install process is naural for beginners and advanced users alike.

Package Manager

By default, SongbirdOS comes with Pacman and SPM. Pacman is the Arch Linux package manager that's used for official reposiory packages. SPM is the Songbird Package Manager. It's used instead of Yay or Paru as an AUR helper. This allows you to install packages from the main repository or the Arch User Repository.


If you would rather use Yay or Paru over SPM then thsi can easily be done in the installer. Simply change the option under Package Manager to your desired choice.

GUI

SongbirdOS has the option to choose from a wide range of GUI environments. The default environment on the live media is SDE, the Songbird Desktop Environment, it's extremely customisable with the ability to bind global hotkeys. And, it's Wayland-based ensuring that it'll be supported more and more going forwards.
