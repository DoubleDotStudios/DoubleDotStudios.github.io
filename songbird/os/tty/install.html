TTY Install

A TTY install is not recommended for most users.


1 Pre-installation

1.1 Obtaining the ISO

Visit the Download page and download the ISO file and the sha512 checksum file.

1.2 Verifying the ISO

It is recommended that you verify the ISO to check for tampering or corruption.

To verify the ISO have the checksum file in the same directory and run:

$ sha512sum --check SongbirdOS-VERSION.iso.sig

1.3 Preparing the Live Media

The official way to install SongbirdOS is via a USB flash drive. To do so use a GUI such as BalenaEtcher, Rufus, Disk Utility. Or use the dd command:

# dd bs=4M if=path/to/SongbirdOS-VERSION.iso of=/dev/sdaX conv=fsync oflag=direct status=progress
